✨ Built as part of Machine Learning Internship – Future Interns

---

🚀 Project Overview

This project focuses on automating the candidate screening process by comparing resumes with job descriptions.
It helps recruiters quickly evaluate candidate suitability and identify skill gaps efficiently.

---

🎯 Objectives

- Analyze resumes using Machine Learning techniques
- Compare candidate skills with job requirements
- Generate a match score (%)
- Identify matched and missing skills
- Assist in faster hiring decisions

---

🧠 Machine Learning Approach

- Converted textual data into numerical form using TF-IDF Vectorization
- Applied Cosine Similarity to measure similarity between resume and job description
- Used set operations to extract matched and missing skills

---

⚙️ System Workflow

1. Input resume text
2. Input job description
3. Convert text into vectors using TF-IDF
4. Compute similarity score
5. Extract matching skills
6. Identify missing skills
7. Display final results

---

📊 Features

- Resume and job description comparison
- Match score calculation (in percentage)
- Skill matching system
- Skill gap identification
- Lightweight and efficient implementation
